## Description
A complete Library Management System that allows users to manage movies, series, games, comics/manga, books. Designed for easy tracking and organization of library resources.

## Features
- Add, update, and remove movies, series (soon: games, comics/manga, books)
- 
- Simple and interactive user interface

## Tech Stack
- Python
- PySide6 (for GUI)
- SQLite3 (for database)
- requsts (for fetching APIs)
- BS4 (for scrape websites)

## Installation / Setup

1.git clone https://github.com/a8392280-web//Libirary-all-in-one.git
2.cd library-project
3.pip install -r requirements.txt
4.python main.py
